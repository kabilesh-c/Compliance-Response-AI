# PharmaOS Basic Evaluation Questionnaire

**Submitted by:** Regional Healthcare Network  
**Date:** March 2026  
**Purpose:** Initial vendor evaluation for pharmacy management system

---

## Questions

1. What type of system architecture does PharmaOS use, and how do users access the platform?

2. How does the platform track medicine inventory levels, and what information is stored for each medicine batch?

3. What user roles are supported by the system, and how are access permissions managed?

4. How does PharmaOS handle prescription verification before medications are dispensed?

5. Does the platform support mobile application access for pharmacists to review prescriptions remotely?

6. What types of operational reports can be generated from the system?

7. How are inventory adjustments recorded and tracked within the platform?

8. What authentication mechanisms are used to control access to the system?
